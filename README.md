# Blinkit Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard created to analyze Blinkit sales data. It provides insights into sales, customers, delivery performance, and payment methods.

## Dashboard Features
- Total Sales
- Total Orders
- Sales by Year
- Sales by Customer
- Sales by Delivery Partner
- Sales by Payment Method
- Delivery Status Analysis
- Interactive Filters (Slicers)

## Tools Used
- Power BI
- Microsoft Excel
- DAX
- Power Query

## Dataset
The dataset contains:
- Customer Details
- Orders
- Delivery Information
- Payment Methods

## Key Insights
- Total Sales: 11.01M
- Total Orders: 5K+
- Most orders are delivered On Time.
- Cash and UPI are the most used payment methods.
- Sales trends can be analyzed year-wise.
